路由守卫逻辑

未连接钱包 → 只能访问 Landing Page，其余路由重定向到 /
已连接 + 未注册 → 弹出注册流程（邮箱验证），完成后进入 Dashboard
INACTIVE → 可访问 Dashboard、Deposit，其余功能页提示"请先存入押金"
ACTIVE → 全部页面可访问
SUSPENDED → 可访问 Dashboard、Billing（支付），其余功能页提示"请先结清账单"
